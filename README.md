# Identity_Theft_on_Telegram


Purpose: Messaging platforms like Telegram have become key marketplaces for stolen personally identifiable information (PII), transforming how such data is organized, distributed, and monetized. Guided by Routine Activity Theory, we explore: (1) how Telegram’s architecture enables the convergence of offenders, targets, and weak guardianship; (2) which types of data are most frequently shared; and (3) how victimization patterns vary by geography and institution.This research addresses a critical gap in understanding how surface -accessible messaging platforms have transformed identity theft operations, moving beyond darknet-focused cybercrime studies.
